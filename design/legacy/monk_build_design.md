# 武僧 Build 设计稿

## 规则源

- 本文档记录 `武僧（体术）` 的职业设计，并以 `design/physical_class_core_skill_design.md` 为规则源。
- 当前职业成长按 `low / mid / high` 三阶子职业结构实现，不再采用旧 `Lv2/Lv4` 分支选择。

## 三阶结构

| 阶段 | 授予 Feat | 技能槽 | 规则定义 |
| --- | --- | --- | --- |
| `low` | `武僧修行` + `连击` | `basic_attack_slot` + `core_slot` | 获得徒手打击；徒手打击命中后有 `50%` 概率对同一目标追加 1 次额外攻击，额外攻击不再触发连击。 |
| `mid` | `开放手` | `mid_slot` | 获得震劲掌，`CD3`；对当前目标发动徒手打击，命中后目标强韧豁免失败则 `STUN 1` 回合。 |
| `high` | `明镜止水` | `high_slot` | 获得高阶救场能力，恢复生命并清除控制或负面状态。 |

## 实现映射

| 能力 | Feat ID Symbol | Skill ID | Runtime |
| --- | --- | --- | --- |
| 徒手打击 | `monk_training` | `80003011` | 主动，基础徒手攻击 |
| 连击 | `monk_martial_arts` | `80003101` | 被动，命中后 50% 额外攻击 |
| 震劲掌 | `monk_open_hand` | `80003013` | 主动，命中后强韧豁免控制 |
| 明镜止水 | `monk_harmonize` | `80003015` | 主动，恢复 + 净化 |

## 语义边界

- 连击是额外攻击，不是附伤。
- 连击追加出的攻击不能再次触发连击。
- 震劲掌复用徒手攻击链，控制只在命中后判定。
