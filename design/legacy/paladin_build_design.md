# 圣骑 Build 设计稿

## 规则源

- 本文档记录 `圣骑（誓约）` 的职业设计，并以 `design/physical_class_core_skill_design.md` 为规则源。
- 当前职业成长按 `low / mid / high` 三阶子职业结构实现，不再采用旧 `Lv2/Lv4` 分支选择。

## 三阶结构

| 阶段 | 授予 Feat | 技能槽 | 规则定义 |
| --- | --- | --- | --- |
| `low` | `圣武训练` + `神圣庇护` | `basic_attack_slot` + `core_slot` | 获得近战基础攻击；每回合第一次受到伤害时获得减伤，承担前线守护定位。 |
| `mid` | `破邪斩` | `mid_slot` | `CD3`，对当前目标发动神圣斩击；命中后追加神圣伤害并驱散目标 1 个正面增益。 |
| `high` | `圣手` | `high_slot` | 治疗生命最低友军，清除负面状态，并承担高阶团队稳定器定位。 |

## 实现映射

| 能力 | Feat ID Symbol | Skill ID | Runtime |
| --- | --- | --- | --- |
| 圣武斩击 | `paladin_training` | `80004011` | 主动，基础近战武器攻击 |
| 神圣庇护 | `paladin_shelter_prayer` | `80004102` | 被动，团队首次受击减伤 |
| 破邪斩 | `paladin_vengeance_smite` | `80004014` | 主动，神圣附伤 + 驱散正面状态 |
| 圣手 | `paladin_lay_on_hands` | `80004013` | 主动，治疗 + 净化 |

## 语义边界

- 破邪斩的驱散发生在命中后。
- 驱散固定为 1 个正面增益，不扩大为全驱散。
- 圣手负责高阶救场，低阶神圣庇护负责前线减伤。
