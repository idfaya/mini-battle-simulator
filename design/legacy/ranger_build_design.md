# 游侠 Build 设计稿

## 规则源

- 本文档记录 `游侠（追猎）` 的职业设计，并以 `design/physical_class_core_skill_design.md` 为规则源。
- 当前职业成长按 `low / mid / high` 三阶子职业结构实现，不再采用旧 `Lv2/Lv4` 分支选择。

## 三阶结构

| 阶段 | 授予 Feat | 技能槽 | 规则定义 |
| --- | --- | --- | --- |
| `low` | `游侠训练` + `猎人印记` | `basic_attack_slot` + `core_slot` | 获得远程基础攻击；命中时给目标附加唯一标记，再次攻击自己的标记目标时造成额外伤害。 |
| `mid` | `狩猎指引` | `mid_slot` | `CD3`，始终作用于当前标记目标；若当前没有标记目标则不应施放。 |
| `high` | `箭雨` | `high_slot` | 对标记目标进行多段追猎：远程基础攻击追加第二击，命中标记目标时造成额外伤害。 |

## 实现映射

| 能力 | Feat ID Symbol | Skill ID | Runtime |
| --- | --- | --- | --- |
| 猎弓射击 | `ranger_training` | `80005011` | 主动，基础远程攻击 |
| 猎人印记 | `ranger_hunter_mark` | `80005101` | 被动，命中后唯一标记 |
| 狩猎指引 | `ranger_hunter_shot` | `80005013` | 主动，标记目标追击 |
| 箭雨 | `ranger_hunter_mastery` | `80005109` + `80005108` | 被动，多段追猎 |

## 语义边界

- 标记归属绑定游侠自身。
- 同一名游侠同时只维持 1 个标记，新标记会覆盖旧标记。
- 箭雨不是无条件全屏清场，而是围绕标记目标的多段追猎。
