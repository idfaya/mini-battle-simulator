# 战士职业设计

## 设计目标

- 本文档独立记录战士的职业设计，不依赖旧版总设计文档。
- 战士成长统一由 Feat 承载，不使用“槽位解锁”作为额外抽象。
- 设计目标是贴近 D&D 5e 的核心体验：稳定武器攻击、前线站场与清晰的单主动技能表达。
- Feat 尽量复用 `基础武器攻击` 这一核心动作，避免引入额外复杂状态机。

## 设计原则

- Lv1 给职业底盘，确保战士开局就有明确身份。
- Lv2 固定给 `额外攻击`，先建立 Fighter 的基础动作频率。
- Lv3 只做一次主动技能分流：`动作激增` 或 `护卫`，保证整套 Build 始终只有一个主动技能。
- Lv4 再补一层被动分流：`精准攻击` 或 `反击战法`。
- Lv5 用终盘节点收束路线：爆发线拿 `横扫攻击`，守线拿 `续战专精`。
- 原 `冠军线`、`压制线` 与多主动技能方案整体移除。
- Feat 文案优先使用离散规则表达，如额外攻击、额外骰、AC、减伤、每回合 1 次、每场战斗 1 次。

## 战士 Lv1-Lv5 Build 表

| 等级 | 类型 | 名称 | 效果描述 | Build 方向 |
| --- | --- | --- | --- | --- |
| Lv1 | 固定 | 战士训练 | 获得基础武器攻击，对单体敌人进行一次标准近战武器攻击。 | 建立战士所有后续构筑的统一母技能。 |
| Lv1 | 固定 | 二次生命 | 每场战斗 1 次，生命首次降到一半及以下时，回复 `1d10 + 等级` 生命。 | 建立 Fighter 的基础续战能力和前线容错。 |
| Lv2 | 固定 | 额外攻击 | 每回合第一次基础武器攻击后，再追加 `1` 次基础武器攻击。 | 先确立战士的高频基础攻击底盘。 |
| Lv3 | 二选一 | 动作激增 | `CD3`，使用后立刻发动 `1` 次基础武器攻击，目标重新选择；不占用、也不替换该回合原本的那次普通攻击。 | 进入偏主动爆发与补刀的路线。 |
| Lv3 | 二选一 | 护卫 | 获得 `护卫架势`：`CD3`，持续到你下回合开始；期间你和友军被攻击时获得 `AC +2`、减少等同熟练加值的伤害；若攻击者为近战单位，则在该次攻击结算后对其发动 `1` 次基础武器攻击。 | 进入偏防护、拦截与守线的路线。 |
| Lv4 | 二选一 | 精准攻击 | 基础武器攻击忽略目标 `2` 点 AC。 | 作为主动爆发线的稳定命中强化。 |
| Lv4 | 二选一 | 反击战法 | 每回合 `1` 次，被近战攻击指定为目标时，在该次攻击结算后对攻击者发动 `1` 次基础武器攻击。 | 作为护卫路线的被动反打强化。 |
| Lv5 | 二选一 | 横扫攻击 | 基础武器攻击命中主目标后，对另一个敌人追加 `1` 次横扫伤害。 | 作为爆发线的终盘清线强化。 |
| Lv5 | 二选一 | 续战专精 | 二次生命额外再回复 `1d10` 生命。 | 作为守线的终盘续战强化。 |

## 推荐路线

| 路线 | 选择顺序 | 玩法概述 |
| --- | --- | --- |
| 爆发线 | `额外攻击 -> 动作激增 -> 精准攻击 -> 横扫攻击` | 以主动出手、补刀、稳定命中与多目标扩散为核心，强调前压输出。 |
| 守线 | `额外攻击 -> 护卫 -> 反击战法 -> 续战专精` | 以短时防护、受击反打与残局续战为核心，强调前排站桩与保护队友。 |

## 设计备注

- `额外攻击` 现在前置到 Lv2，作为所有战士共享的基础频率强化。
- `动作激增` 与 `护卫` 同层二选一，负责决定整套 Build 唯一的主动技能。
- `精准攻击` 与 `反击战法` 放到 Lv4，作为对前一层路线的被动补强。
- `横扫攻击` 与 `续战专精` 放到 Lv5，分别补足爆发线的清线能力与守线的续战能力。
- `反击战法` 的核心表达是“被近战攻击后登记反打”，不依赖是否命中，但表现上应等待对方本次攻击动作完成。
- `护卫` 的核心表达是“短时抬高前排容错并拦截近战”，通过 `AC +2`、熟练减伤和近战护卫反击来完成，不引入额外仇恨系统。
- 原 `冠军线`、`压制线`、`压制打击`、`压制者`、`钢铁护卫` 与 `战技专精` 均从当前目标设计中移除。
- 本设计中的纯属性成长（如 ASI、基础命中、基础生命）应回到等级底盘，不占用 Feat 位。
