# 战士 Build 设计稿

## 规则源

- 本文档记录 `战士（武技）` 的职业设计，并以 `design/newbattle/physical_class_core_skill_design.md` 为规则源。
- 当前职业成长按 `low / mid / high` 三阶子职业结构实现，不再采用旧 `Lv2/Lv4` 分支选择。

## 三阶结构

| 阶段 | 授予 Feat | 技能槽 | 规则定义 |
| --- | --- | --- | --- |
| `low` | `战士训练` + `反击` | `basic_attack_slot` + `core_slot` | 获得基础近战武器攻击；敌方对你发动近战武器攻击后，无论命中与否，在该次攻击结算后反击 1 次。 |
| `mid` | `护卫` | `mid_slot` | `CD3`，持续到你下回合开始；期间你获得 `AC +2`、固定减伤，且敌方近战攻击你或你护卫的友军后，你都会反击。 |
| `high` | `不屈之风` | `high_slot` | 生命值将降到 `0` 时防止死亡，清除状态并恢复到 `50%` 最大生命值；每场战斗 1 次。 |

## 实现映射

| 能力 | Feat ID Symbol | Skill ID | Runtime |
| --- | --- | --- | --- |
| 基础武器攻击 | `fighter_training` | `80002001` | 主动，基础近战武器攻击 |
| 反击 | `fighter_counter_basic` | `80002104` | 被动 reaction，登记后结算 |
| 护卫架势 | `fighter_guard` | `80002005` + `80002105` | 主动 + 被动护卫反击 |
| 不屈之风 | `fighter_second_wind` | `80002101` | 被动濒死防止 |

## 语义边界

- 反击复用基础武器攻击链，不新建独立伤害公式。
- 反击不递归触发反击。
- 护卫架势才提供团队保护；低阶反击只保护自身。
