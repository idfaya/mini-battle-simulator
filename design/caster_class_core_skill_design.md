# 法系职业核心技设计稿

## 设计目标

- 本文档统一收敛当前法系职业的核心技设计，作为后续 `Feat -> Runtime Skill -> 5e Meta` 实现的规则源。
- 目标是在保留 D&D 5e 风味的前提下，对法系职业做足够简化，使其适配当前自动战斗与轻量技能系统。
- 本文档当前覆盖仓库内已有的四个法系职业：
  - 法师（冰）
  - 术士（火）
  - 牧师（神术）
  - 邪术师（雷）
- 所有能力优先复用统一母技能，避免引入完整法术位、专注槽、法术准备列表与复杂区域状态机。
- 本文档按“低阶 / 中阶 / 高阶”三阶子职业方式拆解现有法系职业。

## 总体原则

- 5e 核心公式保持不变：
  - 攻击型法术继续使用现有法术命中链。
  - 豁免型法术继续使用现有法术豁免链。
  - 伤害、减速、点燃、减伤等效果统一挂在既有战斗结算系统上。
- 法系职业差异优先通过以下方式表达：
  - 元素属性
  - 命中后附加状态
  - 目标优先级
  - 主动技能形态
  - 高阶子职业能力的控场、爆发或救场语义
- 自动战斗场景下，法系职业必须同时定义：
  - 敌方目标选择倾向
  - 友方施放对象选择倾向
  - 何时保留技能、何时立即释放
- 所有成长入口统一由 `Feat` 承载，后续落地时只使用 `grant_skill`、`modify_skill`、`replace_skill`。
- 每个法系职业都拆成三阶子职业，与 `class_system_design.md` §2.4 的技能槽一一对应：
  - `low`：启用 `basic_attack_slot` + `core_slot`（`1 普攻 + 1 核心被动`）
  - `mid`：在 `low` 基础上追加 `mid_slot`（`1 主动` 或 `1 被动`）
  - `high`：再追加 `high_slot`（改造后的旧大招或新设计的高阶能力）
- 高阶能力不强制要求保留原大招形态；可以改写成更适合自动战斗和总纲模板的主动或被动。
- 职业成长不写成固定战斗内等级，而应通过同职业进阶、装备与 Feat 链推进。
- 法系职业的高阶能力不追求翻桌，而是放大该职业主循环：
  - 法师放大控场
  - 术士放大点燃爆发
  - 牧师放大救场与守护
  - 邪术师放大印记与连锁压制

## 职业间区分原则

- 法师与邪术师都涉及"挂状态再兑现"，必须在外延上拉开区分：
  - 法师走 `霜冻 -> 冻结` 路线，主轴是 `无法移动的铺垫状态 / STUN`，不做连锁弹射。
  - 邪术师走 `印记 -> 连锁弹射` 路线，主轴是 `静电印记 + 雷链 / 雷暴`，不再附加减速。
- 术士与法师的群体大招必须在结算路径上拉开：
  - 术士的 `烈焰风暴` 必须围绕 `已点燃` 状态做条件兑现，不做独立无差别全屏高伤。
  - 法师的 `暴风雪` 必须围绕 `已霜冻` 状态做条件兑现，不做独立无差别全屏高伤。
- 牧师是唯一"友方目标决策"为主的法系职业，其能力不应替代另三人的输出与控场职责。

## 字段口径

- 本文档采用 `class_system_design.md` 作为 Class 层字段规则源。
- 本文档中的职业能力对象统一为 `Class 单位`，不再使用独立“角色”口径。
- 阶段字段统一使用 `promotion_stage`。
- `promotion_stage` 的数据值统一为：
  - `low`
  - `mid`
  - `high`
- 展示层可渲染为：
  - `低阶`
  - `中阶`
  - `高阶`
- 技能槽统一使用：
  - `basic_attack_slot`
  - `core_slot`
  - `mid_slot`
  - `high_slot`
- 槽位语义统一为：
  - `basic_attack_slot` = 基础出手技能
  - `core_slot` = 低阶核心能力
  - `mid_slot` = 中阶新增能力
  - `high_slot` = 高阶终局能力
- 职业卡只推动 `Class 单位` 获得或进阶，不直接提供等级提升。
- `等级` 只负责数值成长；`进阶` 只负责同职业阶段推进；`转职` 才改变 `class_id`。
- 项目内 `save_proficiency` 字段并不严格对应 5e 原典的六维豁免，统一按以下口径映射：
  - `fort` 对应抗负面状态（毒、力竭、衰弱、致死打击）
  - `ref` 对应区域伤害与位移类豁免
  - `will` 对应控制类豁免（魅惑、恐惧、`STUN`、神志被夺）
- 法系职业 `Speed` 字段的影响范围统一约束为：
  - 影响 `initiative` 排序与近战进入近身的行动消耗
  - 不影响纯远程位移与法术施放距离

## 母技能映射

- 法师：寒霜射线
- 术士：基础火焰法术
- 牧师：神圣火花
- 邪术师：邪能冲击

## 三阶子职业结构

- 所有法系职业都采用统一的 `low -> mid -> high` 三段结构：
  - `low`：建立职业辨识度，只启用 `basic_attack_slot + core_slot`
  - `mid`：在 `low` 基础上追加 `mid_slot`
  - `high`：在 `mid` 基础上追加 `high_slot`
- 推荐授予顺序：
  - 初始获得 `low`
  - 后续通过同职业进阶或关键 Feat 晋升到 `mid`、`high`
- 设计目的：
  - 开局立即建立法系职业差异
  - 中期补足职业差异和可观察联动
  - 后期补足职业高潮与战场控制感
  - 保持自动战斗下的施法逻辑可理解、可预测
  - 让高阶能力服务总纲，而不是反向改写总纲

## 四职业三阶规格

### 法师

- 5e 画像：`primary_ability = int`，`spell_ability = int`，`armor_formula = robe_dex`，`save_proficiency = fort / will`（wizard 位）。

| 阶段 | 能力包 | 规则定义 |
| --- | --- | --- |
| `low` | 寒霜射线 + 寒霜迟滞 | 对单体敌人进行一次标准远程冰系法术攻击；你的寒霜法术命中后，目标进入 `霜冻`：无法移动，但仍可进行远程攻击和释放技能。 |
| `mid` | 冻结新星 | `CD3`，十字范围冰系伤害；已处于 `霜冻` 的目标转为 `冻结 1` 回合，未处于 `霜冻` 的目标改为施加 `霜冻`。 |
| `high` | 暴风雪 | `主动技能`，`CD5`；对全体敌人造成基础冰系伤害；命中时若目标已处于 `霜冻`，则伤害额外 `+1d8` 并刷新 `霜冻`；命中时若目标尚未 `霜冻`，则仅施加 `霜冻`，不附加额外伤害。本技能不与 `雷暴` 共享状态来源。 |

### 术士

- 5e 画像：`primary_ability = int`，`spell_ability = int`，`armor_formula = robe_dex`，`save_proficiency = will`（sorcerer 位）。

| 阶段 | 能力包 | 规则定义 |
| --- | --- | --- |
| `low` | 火焰弹 + 余烬点燃 | 对单体敌人施加一次标准火焰法术豁免对抗；火焰法术使目标未通过豁免后，为其附加燃烧伤害；目标已处于燃烧状态时不重复施加，仅刷新持续时间。 |
| `mid` | 灰烬爆燃 | `CD3`，对当前目标施放一次强化火焰法术；若目标已被点燃，则额外造成火焰伤害。 |
| `high` | 烈焰风暴 | `主动技能`，`CD5`；对全体敌人造成基础火焰伤害；命中时若目标已被点燃，则额外造成 `1d8` 火焰伤害并延长燃烧 `1` 回合；命中时若目标尚未被点燃，则按 `余烬点燃` 规则为其附加燃烧。本技能不消耗 `余烬点燃` 在当回合的额度。 |

### 牧师

- 5e 画像：`primary_ability = str`，`spell_ability = wis`，`armor_formula = medium_capped`，`save_proficiency = will`。

| 阶段 | 能力包 | 规则定义 |
| --- | --- | --- |
| `low` | 神圣火花 + 神恩庇护 | 神圣火花对单体敌人施加一次标准远程神术豁免对抗；若目标为友军，则改为回复 `1d8` 生命。神恩庇护每回合 `1` 次，全队共享一次：本回合第一次有友军被攻击命中时，该次伤害减少 `1d6`。 |
| `mid` | 治愈之言 | `CD3`，为生命值最低的友军回复生命，并优先解除其 `1` 个负面状态。 |
| `high` | 圣域祷言 | `主动技能`，`CD3`，持续 `2` 回合；期间我方全体 `AC +1`，且每个友军每回合第一次受到的伤害减少 `1d6`；与 `神恩庇护` 同回合触发时，仅取较高的减伤值，不叠加。 |

### 邪术师

- 5e 画像：`primary_ability = int`，`spell_ability = int`，`armor_formula = robe_dex`，`save_proficiency = ref / will`（warlock 位）。

| 阶段 | 能力包 | 规则定义 |
| --- | --- | --- |
| `low` | 邪能冲击 + 静电印记 | 对单体敌人进行一次标准远程雷系法术攻击；邪能冲击命中后为目标附加 `静电印记`，持续到目标下回合开始前；同来源重复施加时只刷新持续时间。 |
| `mid` | 雷链 | `CD3`，对当前目标造成雷电伤害，并额外弹射 `1` 名敌人；带有 `静电印记` 的目标优先成为弹射目标。 |
| `high` | 雷暴 | `主动技能`，`CD5`；对全体敌人造成基础雷电伤害；命中时若目标带有 `静电印记`，则额外造成 `1d8` 雷电伤害并清除该目标的印记；命中时若目标尚未带印记，则按 `静电印记` 规则为其挂上印记。本技能不附加 `Speed` 修正，不与 `暴风雪` 共享减速来源。 |

## 职业核心技

### 法师：寒霜迟滞

#### 职业核心技

- `寒霜迟滞` 的职业定位是稳定控速与冻结铺垫。
- 法师的主循环不是纯伤害，而是先挂 `霜冻`，再用中阶与高阶能力兑现控制收益。

#### 触发规则

- 触发条件：你的寒霜法术命中目标后。
- 结算时机：本次伤害结算完成后，给目标附加 `霜冻`。
- 状态定义：`霜冻` = 无法移动，但仍可进行远程攻击和释放技能；`冻结` = `STUN 1` 回合。
- 持续时间：`霜冻` 默认持续 `2` 回合；同来源重复施加时只刷新持续时间。
- 次数限制：不限制次数，但同来源重复施加时只刷新持续时间。

#### 目标选择倾向

- 普攻优先攻击高威胁前排、快要出手的敌人或高机动目标。
- 中阶 `冻结新星` 优先在十字范围内能覆盖 `2` 名及以上敌人，或能命中已 `霜冻` 的高威胁目标时释放。
- 高阶 `暴风雪` 已固定为 `主动技能`，`CD5`；应优先在敌方仍有 `3` 名及以上存活单位、或场上有 `2` 名及以上已 `霜冻` 目标时释放。
- 若场上已有 `霜冻` 的高威胁目标，优先持续压制该目标。

#### 数值口径

- `寒霜迟滞` 改为 `霜冻` 状态，不再依赖 `Speed`。
- `冻结新星` 建议 `CD3`，伤害低于纯输出职业同档主动，但附带稳定控制威胁。
- `暴风雪` 的强度来源应以群体 `霜冻` 与状态联动为主，而不是极端高伤。

#### 实现备注

- 法师优先复用寒霜射线母技能，不另造独立小冰弹体系。
- `霜冻` 建议建模为标准状态，供中阶与高阶能力检查“目标是否已霜冻”。
- 自动战斗 AI 需要感知敌方即将行动或高机动单位，避免冰系控制浪费在无关目标上。

### 术士：余烬点燃

#### 职业核心技

- `余烬点燃` 的职业定位是稳定挂 DOT，并围绕点燃目标兑现条件爆发。
- 术士的主循环是“先点燃，再爆燃”，而不是每个技能都做孤立直伤。

#### 触发规则

- 触发条件：你的火焰法术使目标未通过豁免后。
- 次数限制：不限制次数；目标已处于燃烧状态时不重复施加，只刷新持续时间。
- 结算时机：本次法术伤害结算后，为目标附加燃烧状态。
- 持续时间：到目标下回合结束。

#### 目标选择倾向

- 普攻优先攻击尚未点燃的高威胁目标。
- 中阶 `灰烬爆燃` 优先攻击已被点燃的目标。
- 高阶 `烈焰风暴` 已固定为 `主动技能`，`CD5`；应优先在敌方数量较多或场上已有 `2` 名及以上点燃目标时释放。
- 若敌方只剩单体高血量目标，则优先维持“点燃 -> 爆燃”的单点循环。

#### 数值口径

- DOT 建议起步为 `1d4`，避免点燃本身成为主要伤害来源。
- `灰烬爆燃` 的额外收益建议固定为“已点燃目标额外 `1d8`”。
- `烈焰风暴` 的核心收益应是“点燃联动的群体兑现”，而不是单纯做成无脑高伤清屏。

#### 实现备注

- 点燃状态应带来源标记，避免多术士互相覆盖时语义混乱。
- `余烬点燃` 建议作为标准 DOT 状态，后续所有术士技能只检查“是否已点燃”。
- 自动战斗 AI 必须优先识别已点燃目标，否则主动技能会频繁打空联动价值。

### 牧师：神恩庇护

#### 职业核心技

- `神恩庇护` 的职业定位是持续稳场和第一时间保护队友。
- 牧师的主循环不是单纯奶量，而是“减伤托底 + 关键抬血 + 团队圣域”。

#### 触发规则

- 触发条件：每回合第一次有友军被攻击命中时。
- 结算时机：在该次伤害结算时直接减少伤害。
- 次数限制：每回合 `1` 次，且全队共享，不按"每个友军各享一次"实现。
- 作用目标：默认作用于本回合按时间顺序第一个吃到攻击伤害的友军。

#### 目标选择倾向

- 普攻在团队血线稳定时，优先攻击高威胁敌人或不死/邪恶单位。
- 中阶 `治愈之言` 优先施放给生命值最低或身上有关键负面状态的友军。
- 高阶 `圣域祷言` 已固定为 `主动技能`，`CD3`，持续 `2` 回合；优先在己方多人半血或敌方准备爆发时释放，不应变成长时常驻光环。
- 若团队血量稳定且无紧急救场需求，牧师应优先保留主动给下一次危险波峰。

#### 数值口径

- `神恩庇护` 建议固定为 `1d6` 减伤。
- `治愈之言` 建议为单体高质量治疗，并附带 `1` 个负面清除。
- `圣域祷言` 的强度应控制在短时团队庇护，不建议做成长时常驻光环。

#### 实现备注

- 牧师是唯一明确需要同时设计友军目标选择与施法时机的法系职业。
- `治愈之言` 建议内置“生命最低友军”选择，不额外增加自动战斗的目标判断负担。
- `圣域祷言` 建议建模为团队短时 buff，而不是场地区域效果。

### 邪术师：静电印记

#### 职业核心技

- `静电印记` 的职业定位是短时标记与邪能连锁增幅。
- 邪术师的主循环是“先挂印记，再用雷链与雷暴收束印记收益”。

#### 触发规则

- 触发条件：你的 `邪能冲击` 命中后。
- 次数限制：不限制次数；同来源重复施加时只刷新持续时间。
- 结算时机：伤害结算后给目标挂上 `静电印记`。
- 持续时间：直到目标下回合开始前。
- 额外收益：你本回合第一次对带有印记的目标造成伤害时，额外造成 `1d6` 雷电伤害；该额外收益每回合每目标只结算 `1` 次。

#### 目标选择倾向

- 普攻优先攻击尚未带有印记的高威胁目标。
- 中阶 `雷链` 优先在敌方存在相邻或密集目标时释放，并优先命中已带印记目标。
- 高阶 `雷暴` 已固定为 `主动技能`，`CD5`；优先在敌方人数较多或场上已经存在多个印记目标时释放，并维持"挂印记 -> 兑现印记"闭环。
- 单体残局中，优先维持“邪能冲击挂印记 -> 同目标兑现额外雷伤”的稳定循环。

#### 数值口径

- `静电印记` 的额外雷伤建议固定为 `1d6`。
- `雷链` 建议从 `1` 次弹射起步，后续成长再增加弹射数。
- `雷暴` 的强度应来自"印记额外伤害 + 群体基础雷伤"，不再叠加 `Speed` 修正，避免与 `暴风雪` 同质化。

#### 实现备注

- `静电印记` 必须带来源标记，避免多邪术师互相冲突。
- `雷链` 建议优先复用单体母技能结算，再做额外弹射。
- 自动战斗 AI 需要在“挂印记”和“兑现印记”之间保持闭环，否则邪术师会频繁打散输出。

## 风险点

- 法师
  - 若把 `冻结新星` 和 `暴风雪` 都做成高伤强控，会变成压过所有法系的万能职业。
  - 若 `寒霜迟滞` 不限制同来源刷新规则，可能在多法师时把所有敌人压死在 `Speed` 下限。
- 术士
  - 若 `点燃` 伤害本体过高，会使"点燃本身"压过中阶与高阶收益，让 `灰烬爆燃` 与 `烈焰风暴` 形同摆设。
  - 若 `烈焰风暴` 同时绕过点燃额度并允许重复点燃，会形成"烈焰风暴 -> 全场点燃 -> 下回合再次烈焰风暴"的滚雪球。
- 牧师
  - 若同时给过高奶量、清 debuff、群体减伤，容易在自动战斗中成为绝对最优后排。
  - 若 `神恩庇护` 与 `圣域祷言` 的减伤可叠加，会出现"两层 1d6 减伤"覆盖关键攻击的强度溢出。
- 邪术师
  - 若 `静电印记` 无来源隔离，多人同职业时会出现覆盖与重复兑现问题。
  - 若 `雷暴` 仍附加减速，会与法师 `暴风雪` 同质化，必须严格按"邪术师不再附加 `Speed` 修正"约束执行。
