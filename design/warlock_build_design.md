# 邪术师职业设计

## 设计目标

- 本文档记录 `邪术师（雷）` 的职业设计，并以 `design/newbattle/caster_class_core_skill_design.md` 为规则源。
- 邪术师成长统一由 `Feat` 承载，不引入完整法术位、召唤物、契约武器切换或复杂长链施法状态机。
- 邪术师主轴是 `邪能冲击 -> 静电印记 -> 雷链 -> 雷暴`，重点表达短时印记、连锁弹射和印记兑现。
- 本文档使用三阶子职业结构：`low / mid / high`。

## 字段口径

- `promotion_stage = low / mid / high`。
- `basic_attack_slot`：邪能冲击。
- `core_slot`：静电印记。
- `mid_slot`：雷链。
- `high_slot`：雷暴。
- 5e 画像：`primary_ability = int`，`spell_ability = int`，`armor_formula = robe_dex`，`save_proficiency = ref / will`（warlock 位）。

## 三阶规格

| 阶段 | 能力包 | 规则定义 |
| --- | --- | --- |
| `low` | 邪能冲击 + 静电印记 | 对单体敌人进行一次标准远程雷系法术攻击；邪能冲击命中后为目标附加 `静电印记`，持续到目标下回合开始前；同来源重复施加时只刷新持续时间。 |
| `mid` | 雷链 | `CD3`，对当前目标造成雷电伤害，并额外弹射 `1` 名敌人；带有 `静电印记` 的目标优先成为弹射目标。 |
| `high` | 雷暴 | `主动技能`，`CD5`；对全体敌人造成基础雷电伤害；命中时若目标带有 `静电印记`，则额外造成 `1d8` 雷电伤害并清除该目标的印记；命中时若目标尚未带印记，则按 `静电印记` 规则为其挂上印记。本技能不附加 `Speed` 修正，不与 `暴风雪` 共享减速来源。 |

## 核心循环

- `邪能冲击` 负责稳定远程出手。
- `静电印记` 负责短时标记目标。
- `雷链` 负责围绕印记目标进行中阶弹射。
- `雷暴` 负责群体范围内兑现印记收益，并给未印记目标补挂印记。

## 触发规则

- `静电印记` 在 `邪能冲击` 命中后触发。
- `静电印记` 不限制每回合施加次数；同来源重复施加时只刷新持续时间。
- `静电印记` 持续到目标下回合开始前。
- 你本回合第一次对带有印记的目标造成伤害时，额外造成 `1d6` 雷电伤害；该额外收益每回合每目标只结算 `1` 次。
- `雷暴` 命中带印记目标时额外 `+1d8` 雷电伤害并清除该目标印记。

## 目标选择倾向

- 普攻优先攻击尚未带有印记的高威胁目标。
- `雷链` 优先在敌方存在密集目标时释放，并优先命中已带印记目标。
- `雷暴` 优先在敌方人数较多，或场上已经存在多个印记目标时释放。
- 单体残局中，优先维持 `邪能冲击挂印记 -> 同目标兑现额外雷伤` 的稳定循环。

## 数值口径

- `静电印记` 的额外雷伤固定为 `1d6`。
- `雷链` 从 `1` 次弹射起步，后续成长再增加弹射数。
- `雷暴` 的强度来自印记额外伤害和群体基础雷伤，不附加 `Speed` 修正。

## 实现备注

- `静电印记` 必须带来源标记，避免多邪术师互相覆盖。
- `雷链` 建议优先复用单体母技能结算，再做额外弹射。
- 自动战斗 AI 需要维持 `挂印记 -> 兑现印记` 闭环，否则邪术师会打散输出。

## 风险点

- `静电印记` 无来源隔离时，多邪术师会出现覆盖与重复兑现问题。
- `雷暴` 不能附加减速，否则会和法师 `暴风雪` 同质化。
- 邪术师不做稳定控制主轴，避免抢法师定位。
